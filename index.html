Diese Seite enthält die Vorlesungsfolien für Web-Engineering 2 an der DHBW Karlsruhe.

Um auch Code-Beispiele nutzen zu können, müssen die Folien lokal von einem Webserver ausgeliefert werden. 

    $ git clone https://github.com/Qwiri/WebEng2-Community-Fork
    $ cd WebEng2-Community-Fork
    $ npm ci
    $ npm run start

Hinweis: Es handelt sich um einen Fork des originalen Repos mit folgenden Erweiterungen:

  • Theme Switcher
  • Startseite
  • CI/CD
